Dark Mode

To enable switching betweeen a dark and light mode I defined the colour properties for every element using CSS custom properties. The switch between light and dark modes is controlled by changing the " colour-theme " attribute. Depending on its value (either " light" or " dark" ), different sets of CSS custom properties are applied through attribute selectors

CSS - github/cv/file2
  /*Light And Dark Mode*/    
   [  colour-theme  =  "dark"  ] {    
         --code-block-colour  :   #d04949  ;    
         --background-colour  :   rgb  (  147  ,   147  ,   147  );    
   }    
     
   [  colour-theme  =  "light"  ] {    
       --code-block-colour  :   #f7df1e  ;    
   }
